PROJECT TITLE: AutoComplete
PURPOSE OF PROJECT: Be awesome
VERSION or DATE: 5/14/2018
HOW TO START THIS PROJECT: Run AutocompleteDriver.java
AUTHORS: Viktor Kelemen
USER INSTRUCTIONS: input one of two test files: test1.txt or test2.txt (or
your own txt of words, one word per line)
Then input any prefix your heart desires. If the prefix exists in the
input word list, the autocompleted words will appear in alphabetical order.
If the prefix does not exist in the input, an error message will pop up.

---------------------------------FILES----------------------------------
AutocompleteDriver - Contains the main method.

TrieNode - A class that holds Trie Node objects

R_WayTrie - A class for R-way Trie objects. Inspiration from:
https://www.geeksforgeeks.org/trie-insert-and-search/

test1.txt - First set of words to be fed into Trie

test2.txt - Second set of words (from politicians' speeches) to be fed
into Trie
